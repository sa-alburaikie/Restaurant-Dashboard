# Restaurant Management System

## 📌 Overview
A full-stack web-based restaurant management system built using Laravel and MySQL.  
It includes both customer-facing features and a powerful admin dashboard for complete restaurant control.

---

## 🚀 Features
### Customer Side:
- Browse restaurant menu
- Add items to cart
- Place food orders
- Table reservation system
- View weekly offers and discounts

### Admin Dashboard:
- Manage users and roles
- Manage menu items and pricing
- Manage staff and chefs
- Handle orders and reservations
- Full restaurant control system

---

## 🛠️ Technologies
- Laravel (PHP)
- MySQL
- Livewire
- Blade Templates
- MVC Architecture
- Git & GitHub
